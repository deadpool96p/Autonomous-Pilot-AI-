# Professional Autonomous Vehicle Simulation
*An advanced full-stack AI and Robotics environment built on React, FastAPI, PyTorch, and OpenCV.*

## Overview
This project simulates metropolitan driving scenarios, evolving from a simple Pygame genetic algorithm into a sophisticated platform capable of HD Map generation, Multi-Agent IDM physics, and Computer Vision integration.

## Core Advancements (v2.0)

### 1. High-Definition Urban Mapping (`/utils/import_osm.py`)
Direct import of real-world topologies via **OpenStreetMap (OSMnx)**. The system translates XML into lightweight JSON HD maps containing:
*   Multi-lane geometric boundaries.
*   Dashed and solid road markings.
*   Traffic signs seeded at relevant intersections.
*   Building footprint shading.
*   Includes generating real maps like *Manhattan*, *Kyoto*, and *Paris*.

### 2. Multi-Agent Intelligent Driving Model (IDM) 
Dynamic city agents now exhibit logical traffic behaviors:
*   `NPCCar`: Analyzes velocity and distance to lead vehicles to calculate a safe braking threshold, forming natural traffic jams and shockwaves instead of simplistic ray-cast collision avoidance.
*   `Pedestrian`: Extracts topological graph edges to walk accurately along sidewalks and crosswalks.

### 3. Perception Module Integration (`/perception`)
A scaffolded pipeline to transition from direct coordinate reliance to synthetic visual estimation:
*   **OpenCV Lane Detection**: Employs Inverse Perspective Mapping (IPM) to capture a bird's-eye view, isolating lane boundaries via edge detection and polynomial window fitting.
*   **YOLOv8 Classifiers**: Detects both urban dynamic agents (`object_detection.py`) and static regulatory signs (`sign_detection.py`).
*   **Sensor Fusion**: Inverts the pinhole camera geometry to map raw 2D pixel bounding boxes into accurate 3D world vectors used by the planner.

### 4. Advanced Trajectory Planning (`/controls & /planning`)
*   **Frenet Coordinates**: Shifts the coordinate system from Cartesian (X,Y) to Longitudinal/Lateral (s, d) to conform to the curvature of the road network automatically.
*   **PID Controller**: Employs Proportional, Integral, and Derivative tuning to generate fluid steering inputs, keeping the `d` error precisely at 0 seamlessly around tight hairpins.

### 5. Seamless Background Data Engline (`AutoLearningManager`)
*   Scales to capture **10,000 frames per buffer**.
*   **Zero-Interruption Retraining**: Automatically isolates high-quality (collision-free) telemetry generated by successful autonomous navigation, triggering a subprocess PyTorch `PilotNet` retraining phase entirely invisibly to the user.
*   Tracks data separately based on the active city (e.g., a `steering_model_paris.pth`).

## Execution
```bash
# Set up Python backend
cd backend
pip install -r requirements.txt

# Start both servers automatically
cd ..
npm start
```
